# 🏦 Advanced Banking System (C++)

## 🔹 Overview
A console-based banking system developed in C++ using Object-Oriented Programming and file handling. The system supports multiple users with secure login and persistent data storage.

## 🔹 Features
- Account creation with auto-generated account number
- Secure login using PIN
- Deposit and withdrawal operations
- Fund transfer between accounts
- Transaction history tracking (per account)
- Persistent storage using file handling

## 🔹 Technologies Used
- C++
- STL (vector, string)
- File Handling (fstream)

## 🔹 Key Concepts
- OOP (Classes, Encapsulation)
- Static variables for unique account numbers
- File-based persistence
- Modular design
